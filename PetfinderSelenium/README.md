### Description
This repository automates the SignUp and Login functionality of the Petfinder application using Selenium with Java.
It follows the Page Object Model (POM) design pattern to ensure efficient and maintainable test automation.

---

### Features
- Built on the Selenium for enhanced test automation capabilities.
- Written in Java for compatibility and performance.
- Follows the Page Object Model (POM) for modularity and reusability.
- Uses TestNG for test management and reporting.
- JSON data-driven testing for flexibility.

---

### Prerequisites
1. **Java** (version 21 or later).
2. **Maven** installed.
3. **Selenium WebDriver** configured.
4. **TestNG** for running test cases.

---

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```
2. Install dependencies:
   ```bash
   mvn clean install
   ```
3. Ensure Selenium WebDriver is properly configured.

---

### Project Structure
- **src/main/java/com/petfinder/crowdanalyzer/pageObjects/**: Contains page classes for Login, Register, and Home.
- **src/main/java/com/petfinder/crowdanalyzer/utils/**: Utility classes for driver management and common actions.
- **src/test/java/com/petfinder/crowdanalyzer/**: Test classes for login and registration.
- **src/test/resources/**: Test data stored in `dataTests.json`.
- **pom.xml**: Maven configuration file for dependencies.
- **testng.xml**: TestNG configuration file.

---

### How to Run
1. Execute tests:
   ```bash
   mvn test
   ```
2. View the test reports generated by TestNG.

---

### Customization
- Update test data in `src/test/resources/dataTests.json` as needed.
- Modify or add new page classes in `pageObjects` for new features.
